# ArcadeLab

## 프로젝트 구조

```
ArcadeLab/
├── client/          # Unity 클라이언트 프로젝트
│   ├── Assets/
│   ├── Packages/
│   ├── ProjectSettings/
│   └── ...
└── server/          # Node.js 서버 프로젝트
    ├── package.json
    └── ...
```

## 시작하기

### 클라이언트 (Unity)

1. Unity Hub에서 `client` 폴더를 프로젝트로 열기
2. Unity 버전 확인 및 실행

### 서버 (Node.js)

1. server 폴더로 이동
```bash
cd server
```

2. 의존성 설치
```bash
npm install
```

3. 서버 실행
```bash
npm start
```

## 개발 환경

- Node.js: (버전 추가 필요)
